# HCC Comment Shield

Standalone WordPress comment anti-spam powered by the HCC trust service.

## Features

- Shared comment scoring across sites
- Local whitelist and blacklist rules
- Tolerant, Balanced, and Strict modes
- WordPress logs page
- Feedback learning from admin moderation actions
- HCC AI Tips widget in the WordPress dashboard
- Weekly HTML summary email

## Download

- Latest release: <https://github.com/juliansebastien-rgb/HCC-Comment-Shield/releases>

## Notes

The trust service learns from repeated spam text, confirmed spam email domains, exact email addresses, IP addresses, and author URL domains.
